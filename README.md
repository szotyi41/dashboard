# Dashboard
Python dashboard

# Install
```
apt install python3-tk
brew install python3
```

You can install other requiments with:
```pip3 install -r requirements.txt```

# Run
```python3 hud.py```


# Modules

* Speedometer
* Weather
* Graph
  * Average consumption
* Music player
* Timing (Quarter miles 0,402336 km == 40.2336 m)
